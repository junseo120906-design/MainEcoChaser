외국인 유학생과 기숙사생을
위한 환경 인식 개선 게임

이 게임은 외국인 유학생과 기숙사생이 다른 지역의 분리수거 기준을 쉽고 재미있게 익힐 수 있도록 설계된 환경 인식 개선 게임입니다.
캐릭터가 다양한 지역을 여행하며 쓰레기를 올바른 통에 넣는 미션을 수행하고, 지역마다 다른 분리수거 규칙을 자연스럽게 학습합니다.
누구나 즐길 수 있는 러너 스타일 게임으로, 실천보다 인식 개선에
중점을 둡니다.

다채로운 학습 경험

지역별 분리수거 정보를 게임 스테이지로 재구성하여 복잡한 규칙도 직관적으로 학습합니다.

  • 단순한 이론이 아닌 행동 기반 인식 개선
  • 러너형 진행 + 선택형 퀴즈 조합
  • 번역 기능으로 외국인도 손쉽게 참여

차별화된 시스템

사용자 지역 정보를 JSON 데이터와 연동해 현실적인 미션과 정답을 자동 생성합니다.

  • 지역별 규칙을 반영한 몰입형 콘텐츠
  • 게임 종료 후 오답 노트로 약점 보완
  • 누구나 접근 가능한 직관적 인터페이스

전략적 게임성

실시간 러너 플레이에 전략 요소를 더해 교육 효과를 극대화하고 학습 지속성을 높입니다.

지역별 규칙을 반영한 현실적인 콘텐츠와 전략적 게임성을 결합해 사용자에게 몰입감 있는 교육 경험을 제공합니다.
